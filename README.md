# 📖 Overview

This project is an interactive **Power BI Financial Expense Dashboard** designed to analyze personal expenses through meaningful KPIs, charts, and visual insights.

It helps users understand their spending habits, identify high-expense categories, and monitor monthly financial trends.

---

# ✨ Features

- 📊 Interactive Dashboard
- 💰 Expense Analysis
- 📈 Monthly Expense Trends
- 🥧 Category-wise Spending
- 🎯 KPI Cards
- 🔍 Dynamic Filters & Slicers
- 📉 Data Visualization
  
---

# 🛠 Tech Stack

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
